# SHETRENGAW

SHETRENGAW is a two-player strategy game descended from the ancient game
of Chaturanga. It was invented by the Sto — a people who, over six centuries,
transformed from a nomadic society into a world power and back again. The game
encodes both identities: the Igto represent the nomadic Sto, the Mawdige their
rivals. Each session requires both players to inhabit both sides of that history.

The game is asymmetric by design. The two sides have different strengths,
different paths to victory, and a different relationship with the scoring system.
This asymmetry is not a flaw. It is the soul of the game.

## Play Online

Download `game/shetrengaw_v7_stg.html` and open it in any browser.
No installation required. Supports friendly and competitive scoring.
Generates a `.stg` game record at the end of each session.

## Game Records

After completing a session, the game generates a `.stg` file
(a JSON record of results, scores, and key events). If you play
a memorable game, feel free to share it in Discussions.

## Rules Version

Current rules: **v7**. The HTML file and rulebook are both current.
The `.stg` format records the rules version so older records
remain interpretable as the game evolves.
